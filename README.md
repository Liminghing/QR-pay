## 🤝 Contribution

이 프로젝트는 개인 프로젝트로, 기여를 원하신다면 다음 절차를 참고해 주세요:

1. **이슈 확인**: 현재 진행 중인 이슈를 확인하고, 개선할 점이나 제안할 사항이 있다면 알려주세요.
2. **Fork**: 프로젝트를 Fork하여 개인 저장소에서 자유롭게 수정해 주세요.
3. **수정 및 피드백**: 수정한 내용을 프로젝트에 적용하고 싶으시면, Pull Request를 제출해 주세요.
